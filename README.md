# Password Strength Analyzer & Guide

## Overview

This project was developed for the Security Architecture course.

The application evaluates a user's password and determines whether it is Weak, Medium, or Strong based on multiple security criteria.

## Features

- Password strength analysis
- Detects common passwords
- Checks password length
- Checks uppercase letters
- Checks lowercase letters
- Checks numbers
- Checks special characters
- Provides improvement suggestions

## Technologies Used

- Python 3
- Regular Expressions
